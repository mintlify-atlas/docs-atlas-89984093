## Why Capso?

Every macOS screenshot tool worth using costs money. CleanShot X is $29, Cap is $58. They're excellent apps — but a core productivity feature shouldn't be locked behind a paywall.

Capso is the free, open-source answer: a fully native Swift 6 app with feature parity against the best paid tools in the category.

| Feature | CleanShot X | Shottr | Cap | Capso |
|---|---|---|---|---|
| Screenshots | Full | Full | Basic | **Full** |
| Recording | Video + GIF | No | Video + GIF | **Video + GIF** |
| Webcam PiP | Yes | No | Yes | **Yes (4 shapes)** |
| OCR | Yes | Yes | No | **Yes** |
| Annotation | Advanced | Advanced | Basic | **Advanced** |
| Pin to Screen | Yes | Yes | No | **Yes** |
| Beautification | Yes | No | Yes | **Yes** |
| Native Swift | Yes | Yes | No (Tauri) | **Yes (Swift 6)** |
| Open Source | No | No | Partial | **Yes** |
| Price | $29 | $8 | $58 | **Free** |

## Get started

<Steps>
  <Step title="Download and install">
    Grab the [latest signed DMG](https://github.com/lzhgus/Capso/releases/latest) from GitHub Releases, or install with `brew install --cask capso`. See [Installation](/installation) for details.
  </Step>
  <Step title="Grant required permissions">
    On first launch, Capso will prompt for screen recording permission. Camera and microphone are needed for webcam PiP and audio capture. See [Permissions](/permissions).
  </Step>
  <Step title="Take your first screenshot">
    Click the Capso menu bar icon and choose **Area Screenshot**, or use the default shortcut. Drag to select a region, and the Quick Access panel appears instantly.
  </Step>
  <Step title="Explore annotation and recording">
    Open the Annotation Editor from the Quick Access panel to add markup, beautify the image, or extract text. Start a screen recording from the menu bar to capture video or GIF.
  </Step>
</Steps>

<Note>
Capso requires macOS 15.0 (Sequoia) or later. It runs natively on both Apple Silicon and Intel Macs.
</Note>
